# Support

## Getting help

Use GitHub issues for:

- reproducible bugs
- incorrect maintenance scoring
- documentation gaps
- feature requests

Open a bug report with the repository checked, command used, output, and expected result.

## Security issues

Do not report vulnerabilities in public issues. Follow `SECURITY.md`.

## Scope

GitMaintainer estimates public maintenance signals. It cannot determine maintainer intent, private maintenance activity, or project health outside public GitHub data.
