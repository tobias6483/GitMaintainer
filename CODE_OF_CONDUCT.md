# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, and ask clarifying questions when needed.
- Accept responsibility and apologize when a mistake is made.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove comments, reject contributions, or ban participants whose behavior is harmful to the community.

Instances can be reported through the security contact listed in `SECURITY.md`.
